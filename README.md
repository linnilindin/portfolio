# Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. Features scroll-triggered animations, interactive transitions, and responsive design.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Running it locally

```bash
npm install
npm run dev
```

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - React components and sections
- `data/` - Project data and content
- `hooks/` - Custom React hooks
- `public/` - Static assets

## Features

- Scroll-triggered animations
- Responsive design
- Image optimization with Next.js Image
- Performance-optimised animations using CSS transforms
